# Landing model assets

The landing page uses public glTF sample assets as stronger illustrative examples for the 3D generation workflow. These are referenced from the Khronos glTF Sample Assets repository via jsDelivr CDN and are documented here so future workers know why they were chosen.

| Landing role       | Asset        | License | Source                                                                          |
| ------------------ | ------------ | ------- | ------------------------------------------------------------------------------- |
| Product mockup     | Toy Car      | CC0 1.0 | https://github.com/KhronosGroup/glTF-Sample-Assets/tree/main/Models/ToyCar      |
| Hard-surface asset | Boom Box     | CC0 1.0 | https://github.com/KhronosGroup/glTF-Sample-Assets/tree/main/Models/BoomBox     |
| Environment prop   | Lantern      | CC0 1.0 | https://github.com/KhronosGroup/glTF-Sample-Assets/tree/main/Models/Lantern     |
| Packaging model    | Water Bottle | CC0 1.0 | https://github.com/KhronosGroup/glTF-Sample-Assets/tree/main/Models/WaterBottle |

These are not presented to users as generated output claims. They are used as high-quality examples of the kinds of assets a 3D generation workspace should make legible.
